## Acknowledgements and contact
This framework is being developed as part of an interdisciplinary effort to promote FAIR data management within the scientific study of religion. The project includes support from the Association of Religion Data Archives, Center for Open Science, Templeton Religion Trust, and John Templeton Foundation.

### License and data sovereignty
The code and framework for this project are distributed under the MIT License. 

However, please note that the underlying **concept metadata remains the property of the originating authorities**. While this application extracts and flattens this data for exploratory and harmonization purposes, users must respect the individual licensing constraints (e.g., SNOMED Affiliate License, LOINC License, CC BY 4.0) of the respective source ontologies when downloading or using the underlying dataset. For specific links to each source, see the **Data Sources** tab.
